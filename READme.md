# Frontend Workshop Instructions

- Download the zip file and unzip the contents
- Open frontend-workshop folder in your IDE
    - if you are working in VS Code, download the live server extension (Ritwick Dey) for faster iteration
- Navigate to index.html, right click and select "Open with Live Server"
